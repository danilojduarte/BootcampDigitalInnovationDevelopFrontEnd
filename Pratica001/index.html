Falando sobre tags


Tags

( Voltar )

Vamos começar, explicando primeiramente o que seria uma "tag". As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.(Font - Alura - https://www.alura.com.br/artigos/o-que-e-html-suas-tags-parte-1-estrutura-basica).


tag - Html

( Voltar )

A tag html, define o corpo do documento, neste caso, toda linguagem de marcação, toda estrutura da página, deverá esta dentro das tag de abertura e fechamento html.


tag - Head

A tag head,providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.


tag - Body

( Voltar )

A tag Body, Representa o corpo da página. Dentro da tag body, deverá ser adicionado todas informações, toda a estrutura de uma pagina página html, o body será o esqueleto da página.


tag - i

( Voltar )

A tag i,representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo "itálico".


tag - meta

( Voltar )

Esta tag, é responsavel por informações gerais do documento html, informações que não seram exibidas para usuario.


tag - p

( Voltar )

Utilizada geralmente na construção de páragrafos.


tag - div

( Voltar )

Esta tag, poderá ser utilizada em todo o contéudo do site, pois ela faz limitação de determinados conteúdos a serem utilizados. Poderemos considerar que as div's, são os a estrutura do esquema de caixas, que utilizamos em Html.


tag - img

( Voltar )

Através desta tag, o html permiti adicionar imagens ao site.


tag - form

( Voltar )

Utilizando a tag form, poderemos adicionar fomrulários ao nossos projetos.
